# puntofitcba

Proyecto base de Next.js 14 (App Router) para un e-commerce de suplementos con TypeScript, Tailwind CSS, ESLint, Supabase y lucide-react.

## Desarrollo

1. Instala dependencias: `npm install`
2. Completa las variables de entorno en `.env.local`
3. Levanta el proyecto: `npm run dev`

La aplicación quedará disponible en http://localhost:3000.
